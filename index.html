Installation

Two quick ways to use CorePulse:

npm (recommended)

npm install corepulse

Then import it (example for bundlers):

import CorePulse from "corepulse";
CorePulse.init({ interval: 2000 });
CorePulse.start();

CDN / Plain JS

Just include the script and call the global:

<script src="corepulse.js"></script>
<script>
  CorePulse.init({ interval: 2000 });
  CorePulse.start();
</script>

Tip: Open demo.html to see CorePulse collecting metrics in the browser console.

About

CorePulse started as a tiny developer utility to make performance visible without heavy setup. Use it for debugging, demos, or as a base for a custom dashboard.
